# 用户文档

## 项目简介

Jishe Dashboard 是一个硬件数据分析网站，提供：

- SoC 数据大屏（雷达图 / 柱状图 / 散点图）
- SoC 对比页面（2-4 个 SoC 并排对比）
- SoC 详情页（亮点、避雷、搭载设备）
- PC 装机模拟器（按性价比优先）
- 装机方案保存（需登录）

## 快速开始

1. 访问 `/` 查看首页数据大屏。
2. 访问 `/soc` 浏览并筛选 SoC。
3. 访问 `/soc/compare` 进行 SoC 对比。
4. 访问 `/builder` 进行装机选择。
5. 访问 `/login` 登录后可保存方案。
