# User Guide

## Product Overview

Jishe Dashboard provides:

- SoC data dashboard with radar/bar/scatter analytics
- SoC comparison page with side-by-side metrics
- SoC detail pages with highlights, warnings, and supported devices
- PC builder with value-priority hardware selection
- Saved build plans (requires account login)

## Quick Start for Users

1. Open `/` to view the dashboard.
2. Open `/soc` to browse and filter SoCs.
3. Open `/soc/compare` to compare 2-4 SoCs.
4. Open `/builder` to build a PC list by category.
5. Open `/login` to sign in or register before saving builds.
